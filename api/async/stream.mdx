title: Stream utilities
description: Lazy, fluent API for building async data pipelines
---

A lazy, fluent API for building async data pipelines. Operations are not executed until a terminal method (`toArray`, `forEach`, `drain`) is called.

## AsyncStream

### from

Creates a new AsyncStream from an iterable, async iterable, or promise of an iterable.

```typescript
static from<T>(source: StreamSource<T>): AsyncStream<T>
```

#### Parameters

<ParamField path="source" type="StreamSource<T>" required>
  The source data. Can be an iterable, async iterable, or promise that resolves to an iterable.
</ParamField>

#### Returns

<ResponseField type="AsyncStream<T>">
  A new AsyncStream instance.
</ResponseField>

## Transformation methods

### map

Adds a concurrent map step to the pipeline.

```typescript
map<R>(
  mapper: (item: T, index: number) => Promise<R> | R,
  options?: ConcurrencyOptions,
): AsyncStream<R>
```

#### Parameters

<ParamField path="mapper" type="(item: T, index: number) => Promise<R> | R" required>
  Function to transform each item.
</ParamField>

<ParamField path="options" type="ConcurrencyOptions">
  Options for controlling concurrency.
  
  <Expandable title="properties">
    <ParamField path="concurrency" type="number" default="Infinity">
      Maximum number of concurrent operations.
    </ParamField>
    
    <ParamField path="signal" type="AbortSignal">
      Optional AbortSignal to cancel the operation.
    </ParamField>
  </Expandable>
</ParamField>

### filter

Adds a filter step to the pipeline.

```typescript
filter(
  predicate: (item: T, index: number) => Promise<boolean> | boolean,
  options?: ConcurrencyOptions,
): AsyncStream<T>
```

#### Parameters

<ParamField path="predicate" type="(item: T, index: number) => Promise<boolean> | boolean" required>
  Function to test each item. Return `true` to keep the item.
</ParamField>

<ParamField path="options" type="ConcurrencyOptions">
  Options for controlling concurrency.
</ParamField>

### retry

Adds a retry wrapper around the previous step.

```typescript
retry(options?: { times?: number; delay?: number }): AsyncStream<T>
```

#### Parameters

<ParamField path="options" type="object">
  Retry options.
  
  <Expandable title="properties">
    <ParamField path="times" type="number" default="3">
      Maximum number of attempts.
    </ParamField>
    
    <ParamField path="delay" type="number">
      Delay between retries in milliseconds.
    </ParamField>
  </Expandable>
</ParamField>

## Terminal methods

### toArray

Executes the pipeline and collects results into an array.

```typescript
toArray(options?: StandardOptions): Promise<T[]>
```

#### Parameters

<ParamField path="options" type="StandardOptions">
  Execution options.
  
  <Expandable title="properties">
    <ParamField path="signal" type="AbortSignal">
      Optional AbortSignal to cancel the operation.
    </ParamField>
  </Expandable>
</ParamField>

#### Returns

<ResponseField type="Promise<T[]>">
  An array of all results.
</ResponseField>

### forEach

Executes the pipeline and calls `fn` for each result.

```typescript
forEach(
  fn: (item: T, index: number) => Promise<void> | void,
  options?: StandardOptions,
): Promise<void>
```

#### Parameters

<ParamField path="fn" type="(item: T, index: number) => Promise<void> | void" required>
  Function to call for each item.
</ParamField>

<ParamField path="options" type="StandardOptions">
  Execution options.
</ParamField>

### drain

Executes the pipeline, discarding results.

```typescript
drain(options?: StandardOptions): Promise<void>
```

#### Parameters

<ParamField path="options" type="StandardOptions">
  Execution options.
</ParamField>

### reduce

Executes the pipeline and reduces the results.

```typescript
reduce<R>(
  reducer: (accumulator: R, item: T, index: number) => Promise<R> | R,
  initialValue: R,
  options?: StandardOptions,
): Promise<R>
```

#### Parameters

<ParamField path="reducer" type="(accumulator: R, item: T, index: number) => Promise<R> | R" required>
  Function to reduce items to a single value.
</ParamField>

<ParamField path="initialValue" type="R" required>
  Initial accumulator value.
</ParamField>

<ParamField path="options" type="StandardOptions">
  Execution options.
</ParamField>

#### Returns

<ResponseField type="Promise<R>">
  The final reduced value.
</ResponseField>

## Examples

### Basic map and filter

```typescript
import { AsyncStream } from '@temelj/async';

const result = await AsyncStream.from([1, 2, 3, 4, 5])
  .map(async (x) => x * 2)
  .filter((x) => x > 5)
  .toArray();

console.log(result); // [6, 8, 10]
```

### Concurrent processing

```typescript
import { AsyncStream } from '@temelj/async';

const users = await AsyncStream.from(userIds)
  .map(
    async (id) => {
      const response = await fetch(`/api/users/${id}`);
      return await response.json();
    },
    { concurrency: 5 }
  )
  .toArray();
```

### Async iterable source

```typescript
import { AsyncStream } from '@temelj/async';

async function* generateNumbers() {
  for (let i = 0; i < 10; i++) {
    await delay(100);
    yield i;
  }
}

const result = await AsyncStream.from(generateNumbers())
  .map((x) => x * 2)
  .filter((x) => x % 4 === 0)
  .toArray();

console.log(result); // [0, 4, 8, 12, 16]
```

### forEach for side effects

```typescript
import { AsyncStream } from '@temelj/async';

await AsyncStream.from(items)
  .map(async (item) => processItem(item))
  .forEach(async (result) => {
    await saveToDatabase(result);
  });
```

### Reduce to single value

```typescript
import { AsyncStream } from '@temelj/async';

const sum = await AsyncStream.from([1, 2, 3, 4, 5])
  .map(async (x) => x * 2)
  .reduce((acc, x) => acc + x, 0);

console.log(sum); // 30
```

### Drain for side effects only

```typescript
import { AsyncStream } from '@temelj/async';

await AsyncStream.from(notifications)
  .map(async (notif) => {
    await sendEmail(notif);
    return notif;
  })
  .drain();

console.log('All emails sent');
```

### Complex pipeline

```typescript
import { AsyncStream } from '@temelj/async';

const results = await AsyncStream.from(urls)
  .map(
    async (url) => {
      const response = await fetch(url);
      return await response.json();
    },
    { concurrency: 3 }
  )
  .filter((data) => data.isValid)
  .map(async (data) => {
    return await transformData(data);
  })
  .toArray();
```

### With abort signal

```typescript
import { AsyncStream } from '@temelj/async';

const controller = new AbortController();

const promise = AsyncStream.from(items)
  .map(
    async (item) => {
      await delay(100);
      return processItem(item);
    },
    { signal: controller.signal }
  )
  .toArray();

// Cancel after 5 seconds
setTimeout(() => controller.abort(), 5000);

try {
  await promise;
} catch (error) {
  console.log('Stream cancelled');
}
```
